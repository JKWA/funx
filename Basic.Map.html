      @spec add_two_either(integer() | String.t()) :: Funx.Either.t(String.t(), integer())

      Wraps an integer or string in the Either monad and increments it by two if it's an integer.

      Uses Either.right/1 for integers to indicate a successful path, and Either.left/1 for strings to signal an error, avoiding the need for try-catch blocks.

      Examples 

      iex> Basic.Map.add_two_either(3)
%Funx.Either.Right{right: 5}

iex> Basic.Map.add_two_either("Oops, error")
%Funx.Either.Left{left: "Oops, error"}
