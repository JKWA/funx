      View Source Funx.Reader (funx v0.1.0)

      The Funx.Reader module provides an implementation of the Reader monad, which allows for dependency injection by passing a shared environment to functions.

      The Reader monad enables defining computations that read from an environment without explicitly passing the environment around, supporting mapping, binding, and function application.
